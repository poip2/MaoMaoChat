# 前端伪代码（精简版）

## 技术栈
SvelteKit + Svelte 5 + Tailwind CSS

---

## 核心：3个Store + 1个渲染器

```
tabsStore      → 管理所有打开的标签页
documentStore  → 当前显示的文档
settingsStore  → 用户偏好设置

renderer       → Markdown → HTML
```

---

## 主要流程

### 打开文件
```
用户操作 → 文件对话框 → invoke("read_markdown_file") 
    → 得到内容 → renderer渲染 → 创建新标签 → 显示
```

### 保存文件
```
Ctrl+S → invoke("write_markdown_file") → 标记已保存
```

### 编辑模式
```
Ctrl+E → 切换编辑/预览 → 实时更新内容 → dirty标记
```

---

## 标签页管理

```javascript
Tab {
    filePath, fileName
    content        // 原始markdown
    renderedHtml   // 渲染后的HTML
    isEditing      // 编辑模式？
    dirty          // 有未保存更改？
}

addTab()    → 新建或切换到已有
closeTab()  → 关闭并切换到相邻
switchTab() → 保存滚动位置后切换
```

## UI组件（只列核心）


## 快捷键

```
Ctrl+O  打开文件
Ctrl+S  保存
Ctrl+E  编辑/预览切换
Ctrl+W  关闭标签
Ctrl+F  搜索
j/k     Vim滚动
```

## 与后端通信

```javascript
// 调用Rust命令
invoke("read_markdown_file", { path })
invoke("write_markdown_file", { path, content })
invoke("start_watching", { path })

// 监听Rust事件
listen("file-changed", reload)
listen("ai-chunk", updateEditor)
```
