## 通信机制

### 前端 → 后端 (Tauri Commands)
```javascript
// 前端调用
import { invoke } from '@tauri-apps/api/core';
const content = await invoke('read_markdown_file', { path: '/path/to/file.md' });
```

```rust
// 后端处理
#[tauri::command]
fn read_markdown_file(path: String) -> Result<String, String> {
    // 实现...
}
```

### 后端 → 前端 (Events)
```rust
// 后端发送
app.emit("file-changed", &path)?;
```

```javascript
// 前端监听
import { listen } from '@tauri-apps/api/event';
await listen('file-changed', (event) => {
    console.log('文件已更改:', event.payload);
});
```

## 核心流程

### 1. 打开文件流程
```
用户点击打开 → 前端文件对话框 → Tauri命令 → Rust读取文件 
    → 返回内容 → 前端渲染 → 创建标签页 → 更新UI
```

### 2. 文件监视流程
```
Rust监视器检测变化 → 发送事件 → 前端接收 → 重新读取文件 
    → 重新渲染 → 更新标签页内容
```

### 3. AI生成流程
```
用户输入提示 → 前端调用命令 → Rust发送API请求 
    → 流式返回 → 发送chunk事件 → 前端实时更新
```

## 技术栈总结

### 前端
- **框架**: SvelteKit + Svelte 5
- **样式**: Tailwind CSS v4
- **构建**: Vite
- **Markdown**: markdown-it + 插件
- **语法高亮**: highlight.js
- **数学公式**: KaTeX
- **图表**: Mermaid

### 后端
- **框架**: Tauri v2
- **语言**: Rust
- **文件监视**: notify crate
- **HTTP客户端**: reqwest
- **密钥管理**: keyring crate
- **序列化**: serde + serde_json

### 跨平台支持
- macOS (.dmg)
- Windows (.msi)
