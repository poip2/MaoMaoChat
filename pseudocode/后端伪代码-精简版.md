# 后端伪代码（精简版）

## 技术栈
Rust + Tauri v2

---

## 核心：4个命令 + 1个监视器

```
read_markdown_file   → 读取文件
write_markdown_file  → 写入文件
start_watching       → 监视文件变化
ai_generate          → AI生成内容
```

---

## 文件操作

```rust
#[tauri::command]
fn read_markdown_file(path: String) -> String {
    fs::read_to_string(path)
}

#[tauri::command]
fn write_markdown_file(path: String, content: String) {
    fs::write(path, content)  // 原子写入
}
```

---

## 文件监视

```rust
#[tauri::command]
fn start_watching(path: String) {
    // 使用notify crate监视文件
    // 变化时发送事件给前端
    app.emit("file-changed", path)
}
```

## AI功能

```rust
#[tauri::command]
fn ai_generate(prompt: String) {
    // 1. 从密钥链读取API配置
    // 2. 调用OpenAI兼容API
```

## 菜单系统

```rust

    文件 → 打开、粘贴
    编辑 → 撤销、复制、粘贴
    视图 → 查找、主题、缩放
    AI   → Google搜索、自定义提示
```
